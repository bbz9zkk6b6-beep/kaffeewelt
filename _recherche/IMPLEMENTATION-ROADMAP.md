# Implementierungs-Roadmap — Meine kleine Kaffeewelt
**Domain**: meine-kleine-kaffeewelt.de  
**CMS**: Publii | **Erstellt**: 2026-05-28

---

## Phase 1: Foundation (Wochen 1–4)

### Technisches Setup

- [ ] Publii: Passendes Theme wählen (hell, klar, Blog-fokussiert)
- [ ] Custom Domain in Publii konfigurieren
- [ ] SSL/HTTPS aktivieren beim Hosting
- [ ] Deployment-Methode einrichten (FTP, GitHub Pages, Netlify, o.ä.)
- [ ] Publii SEO-Einstellungen konfigurieren:
  - Site Name, Beschreibung, Keywords
  - Open Graph aktivieren
  - Sitemap automatisch generieren
  - Canonical Tags aktivieren
- [ ] robots.txt anlegen
- [ ] Google Search Console: Domain verifizieren + Sitemap einreichen
- [ ] Google Analytics 4 oder Plausible Analytics einbinden
- [ ] Favicon erstellen (☕ Kaffee-Motiv)

### Pflichtseiten erstellen

- [ ] Startseite (mit Teaser der 3 besten Artikel)
- [ ] Über mich (persönliche Geschichte, Foto, E-E-A-T)
- [ ] Impressum (DE-Pflicht: Name, Adresse, E-Mail)
- [ ] Datenschutzerklärung (DSGVO-konform)
- [ ] Kontakt

### Erste Inhalte

- [ ] Pillar-Artikel „Kaffeezubereitung" (1.500 Wörter)
- [ ] Erster Blog-Artikel: „Cold Brew selber machen"
- [ ] „Über mich" Seite fertigstellen
- [ ] Kategorien in Publii anlegen

---

## Phase 2: Expansion (Wochen 5–12)

### Content

- [ ] 2 Artikel/Woche veröffentlichen (laut Content-Kalender)
- [ ] Alle 9 Keyword-Cluster mit mind. 1 Artikel besetzen
- [ ] Interne Verlinkungen bei jedem neuen Artikel prüfen
- [ ] Bilder: WebP-Format, komprimiert, Alt-Tags gesetzt

### SEO

- [ ] Schema Markup für jeden Artikel (Article JSON-LD)
- [ ] HowTo-Schema bei Anleitungen
- [ ] FAQPage-Schema bei Artikeln mit FAQ-Sektion
- [ ] Meta Titles und Descriptions für alle Seiten prüfen
- [ ] Google Search Console: erste Rankings beobachten

### Community & Social

- [ ] Instagram-Profil für den Blog anlegen
- [ ] Pinterest-Profil anlegen (sehr gut für Kaffee-Content)
- [ ] 2–3 Kaffee-Communities beitreten (Reddit r/kaffee, Facebook-Gruppen)
- [ ] Newsletter-Anmeldeformular in Publii einbinden

---

## Phase 3: Scale (Wochen 13–24)

### Content-Tiefe

- [ ] Top-5-Artikel mit niedrigem Traffic aktualisieren/erweitern
- [ ] Pillar-Seiten auf 2.000+ Wörter ausbauen
- [ ] Infografiken für Zubereitungsmethoden erstellen (Canva)
- [ ] Kaffee & Reise-Nische mit ersten 2–3 Artikeln erschließen

### Linkbuilding

- [ ] 3–5 kleine Röstereien kontaktieren: Gastbeitrag oder Erwähnung anbieten
- [ ] DACH-Kaffee-Blogs kommentieren (authentisch)
- [ ] Google Business Profile anlegen
- [ ] In 2–3 Kaffee-Verzeichnissen eintragen

### Analytics

- [ ] Monatliches SEO-Reporting einrichten:
  - Top-10-Rankings in Google Search Console
  - Organischer Traffic-Trend
  - Beste Artikel (Klicks, Impressionen)
  - Content-Gaps aus Search Console (Anfragen ohne Klicks)

---

## Phase 4: Autorität (Monate 7–12)

### Thought Leadership

- [ ] Erste eigene Mini-Studie / Umfrage veröffentlichen
- [ ] „Kaffee-Wissen"-Newsletter aufbauen (Monatlich)
- [ ] 1–2 Kooperationen mit Röstereien (Produkttest, Rezension)
- [ ] Podcast-Gastauftritt bei Kaffee-Podcast

### Technische Optimierung

- [ ] Core Web Vitals auswerten und optimieren
- [ ] Bilder-Lazy-Loading prüfen
- [ ] Breadcrumbs in allen Kategorien aktivieren
- [ ] Hreflang prüfen (falls AT/CH-Variante geplant)

### GEO (AI Search Optimization)

- [ ] Alle Pillar-Artikel mit klar zitierbaren Fakten versehen
- [ ] Tabellen in Vergleichsartikeln einbauen
- [ ] FAQ-Sektionen in Top-10-Artikel einfügen
- [ ] Definitive erste Sätze nach H2-Überschriften

## Quick Wins — Sofort starten

1. **Publii Setup** — Theme, Domain, SSL, Sitemap → ½ Tag
2. **Cold Brew Artikel** — Perfektes Timing (Sommer!) → sofort veröffentlichen
3. **Über mich** — Vertrauen aufbauen → vor erstem Artikel fertig haben
4. **Kaffee-Mythen Artikel** — Viral-Potenzial, einfach zu schreiben → Woche 1–2
5. **Pinterest-Profil** — Kaffee-Infografiken = kostenloser Traffic-Kanal
